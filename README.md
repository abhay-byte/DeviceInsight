**DeviceInsight** is a premium, high-performance system monitoring application for Android. It provides real-time analytics for CPU, RAM, Storage, Network, and Battery usage, presented in a stunning, futuristic "Glassmorphism" interface.

## ‚ú® Key Features

*   **Premium Glass UI**: A sophisticated user interface featuring frosted glass effects, animated cards, and dynamic theme switching.
*   **Real-time Monitoring**:
    *   **CPU**: Live load history with monotonic scrolling graphs.
    *   **Power**: Real-time wattage estimation with adjustable multipliers (0.01x - 100x).
    *   **RAM & Storage**: Instant usage stats with visual circular gauges.
    *   **Network**: Real-time upload/download speeds.
*   **Historical Graphs**: Vico-powered charting for CPU and Power history (last 60 seconds).
*   **Theme Engine**: 5 distinct themes including "Tech-Noir", "Cyberpunk Edge", and "Deep Ocean".

## üõ† Tech Stack

*   **Language**: Kotlin
*   **UI Framework**: Jetpack Compose with a custom Glassmorphism UI system (Haze-powered, with selective Material components)
*   **Architecture**: MVVM + Clean Architecture
*   **Dependency Injection**: Hilt
*   **Asynchrony**: Coroutines & Flow
*   **Charting**: Vico (v2.0.0-alpha.28)
*   **Build System**: Gradle 8.9 (Kotlin DSL)
